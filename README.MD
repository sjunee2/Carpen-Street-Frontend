# Calendar-Todolist

## 실행방법

### Running the development server.

```bash
    pnpm dev
```

### Building for production.

```bash
    pnpm build
```

### Running the production server.

```bash
    pnpm start
```

## 로컬 주소
port: 3000

## 사용툴
Next.js, Redux toolkit, Styled-components, TypeScript
